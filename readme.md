# My Package

A utility package for reusable functions and classes.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

```bash
npm install my-package
```

## Usage

```typescript
import { MyUtil } from "my-package";

console.log(MyUtil.greet("World")); // Hello, World!
console.log(MyUtil.add(2, 3)); // 5
```
